# Proyecto Eien System Kizuna

Este proyecto es un sistema de gestión logística de alta precisión para Kizuna Logística S.A.S. Desarrollado bajo estándares de eficiencia Japonesa. (Kizuna Logísitca S.A.S es una empresa de logísitca ficticia creada para fines académicos).

## Explicación Académica

El proyecto se desarrolla en el marco educativo del SENA (Servicio Nacional de Aprendizaje) en Colombia, como parte del programa de formación en Análisis y Desarrollo de Software (ADSO). El objetivo es aplicar los conocimientos adquiridos en el programa para desarrollar un software que permita mejorar o subsanar las deficiencias de una empresa real o ficticia. En este caso, se ha elegido una empresa ficticia llamada Kizuna Logística S.A.S, la cual se dedica a la gestión logística de alta precisión.
